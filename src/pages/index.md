## Personalpronomen

|              | 1. Person | 2. Person | 3. Person   |
| ------------ | --------- | --------- | ----------- |
| **Singular** | ich       | du        | er, sie, es |
| **Plural**   | wir       | ihr       | sie         |

## Sein und haben

|                 | sein | haben |
| --------------- | ---- | ----- |
| **ich**         | bin  | habe  |
| **du**          | bist | hast  |
| **er, sie, es** | ist  | hat   |
| **wir**         | sind | haben |
| **ihr**         | seid | habt  |
| **sie, Sie**    | sind | haben |

## Artikel und Genus

| Genus      | bestimmter Artikel | unbestimmter Artikel |
| ---------- | ------------------ | -------------------- |
| Maskulinum | **der** Anzug      | **ein** Anzug        |
| Femininum  | **die** Mütze      | **eine** Mütze       |
| Neutrum    | **das** Kleid      | **ein** Kleid        |
